## My process

I approached the challenge by breaking down the webpage into boxes. I created individual classes for the flex behaviour and the cards behaviour that would contain the information. The body of the webpage acts as a flexbox which alligns different information cards over it. The card class allows the creation of a vertical box which displays information.


### Built with

- Semantic HTML5 markup
- CSS custom properties
- Flexbox


### What I learned

This project helped me refine my knowledge of html tags and developing a structured approach towards webpages. I developed a better understanding of how flexbox behaves and alligns items over different parameters of a web page.
